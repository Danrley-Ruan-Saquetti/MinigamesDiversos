ROLETA
Aposte um valor e escolha um número de 0 até 38. Se o número sorteado for aquele que você escolheu, poderá ganhar 5 vezes o valor apostado, porém, as chances são de 2%!

CAÇA NÍQUEIS
Para ganhar dinheiro no Caça Níqueis, deverá pagar uma entrada de R$5,00 para rodá-lo. Após pagar a entrada, rode o níquel e faça um combo de sequência do mesmo valor. 
Os prêmios são: para cada par, ganha-se R$15,00; para cada trinca, ganha-se R$25,00; uma quadra, ganha-se R$30,00; e um perfeito (quíntupla), ganha-se R$50,00.

BINGO
O clássico Bingo! Para jogá-lo, deve-se fazer a inscrição de R$50,00. Você irá competir contra 20 jogadores.
Risque os valores da cartela que foram sorteados e esteja entre os primeiros a completar a cartela para ganhar o prêmio.
Os prêmios são: 1º lugar - R$250,00; 2º lugar: R$200,00; 3º lugar - R$150,00; 4º lugar: R$100,00.
